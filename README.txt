#App para organizar tiempo tareas 

#objetivo de la app:
-ayudarme a estar constantemente realizando tareas de varias tipos
como descanzar , lavar, trabajar, estudiar ,etc.
-darme una indicacion clara de pendientes que tenga
-facil de leer, cuando este de pendejo poder ver la lista y escoger una tarea para hacer

#logica del app

-input de texto para introducir un string a una lista de tareas
-en esta lista de tareas cada item que se agregue va a ser acompa;ado de boton para eliminar
,agregar a todo
-se elimina de lista de tareas y se agrega a lista todo
-lista todo muestra items que se movieron de la lista de tareas y se acompa;an con botones para
regresar a lista de tareas, marcarla como completa en lista todo

#secciones
input
lista de tareas
lista todo
